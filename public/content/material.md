## Key Definitions Quick Ref

- **AoI**: `Δ(t) = t - U(t)` where U(t) is generation time of latest received update
- **PAoI**: Peak AoI, worst-case freshness
- **AoII**: Age of Incorrect Information — time since receiver has been incorrect vs source
- **VoI**: Value of Information — utility / reduction in uncertainty
- **QAoI**: Query AoI — AoI at query instants
- **AoS**: Age of Synchronization
